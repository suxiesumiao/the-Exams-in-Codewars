## Note
* 学的前端，都是js的做法，如果有其他语言方法，注明请
* 我会把自己做得还有票选的高的贴粗来以供参考
* 不会每道题都写一个页面，做这个repo也是为了用下github的issue功能
* 每道题都会注出所属的门类比如 正则 数组 字符串 等
## 期望的格式
* hard: 显示题目的难度，用过的都懂，数字越小难度相对越大
* location: 题目的原始地址
* tags: 题目所属的门类
* examples: 基本每道题都会有例子，输出啥东西，就是参照
* deals: 解法
